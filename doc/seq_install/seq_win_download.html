System Requirements

The current version of Sequin works with the following Windows versions:
  • Windows 95
  • Windows 98
  • Windows 2000
  • Windows NT 4.0+
  • Windows XP
It may work on other versions (Me, NT 3.51), as well.

Downloading Sequin

A single executable program works for all supported Windows platforms.
  1. Download Sequin using your Web browser by clicking this link:
    sequin.win32.exe
  2. Your Web browser may show a dialog box asking if you want to run the program, or save it to disk. Choose Save to disk... and save the file somewhere on your hard drive.
  3. If your Web browser can't download the file for some reason, see Troubleshooting.

Installing Sequin

Follow these instructions to install Sequin on Windows:
  1. Create an installation directory for the program; for example, C:\Program Files\Sequin. This directory may be anywhere on any hard drive in your system.
  2. Move the file sequin.win32.exe to the installation directory.
  3. Run the sequin.win32.exe program by double-clicking on it in Windows Explorer. The program will create a number of files and directories.
  4. If you like, create shortcuts on the desktop or in the Start Menu. ( How to.)
  5. Start sequin by double-clicking the sequin icon in the install directory, on the desktop, or in the Start Menu.
