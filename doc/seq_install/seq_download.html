Troubleshooting

The latest versions of Sequin for Windows, Macintosh, and Unix/Linux are available from the NCBI Web site.

You must first choose whether to download Sequin using your Web browser or using ftp. Generally, downloading with a browser is easier than using ftp. We recommend using ftp only if the browser download does not work.

To download with a browser, please select your operating system:

Windows 95, 98, Me, 2000, NT 4.0+, XP
Macintosh MacOS 9, OS X
Unix Solaris/Sparc, Solaris/Intel, SGI IRIX 5.X, SGI IRIX 6.2 and higher, Linux/x86, and Compaq Tru64 Unix (formerly known as Digital Unix or OSF/1) on Alphas
